# 🚀 Productivity Dashboard

A modern and responsive Productivity Dashboard built using **HTML, CSS, and JavaScript**. It helps users stay organized, focused, and productive with features like task management, daily planning, a Pomodoro timer, live weather updates, motivational quotes, and dark mode.

## ✨ Features

- 📅 **Live Date & Time**
- 🕒 **Analog Clock**
- 🌦️ **Live Weather Widget**
  - Uses Geolocation API
  - Fetches weather using Open-Meteo API
- ✅ **Todo List**
  - Add, delete, mark important, complete tasks
  - Local Storage support
- 📆 **Daily Planner**
  - Hourly scheduling
  - Edit & delete plans
  - Prevents overlapping time slots
  - Highlights the current plan
- 🍅 **Pomodoro Timer**
  - 25-minute work session
  - 5-minute break session
  - Start, Pause, Reset
  - Audio notification when sessions end
- 💬 **Motivation Quotes**
  - Fetches random quotes using Fetch API
- 🎯 **Daily Goals**
  - Add goals
  - Mark as completed
  - Progress tracking
  - Local Storage support
- 🌙 **Dark Mode**
  - Theme persistence using Local Storage
- 📱 **Responsive Design**
- 🎨 **Glassmorphism UI**
